## 1. 开场 / 场景建立

| 效果名 | 难度 | 适合场景 |
|---|---|---|
| Fade 黑屏淡入淡出 | ★ | 章节开头、时间大跳跃（"三年后…"）、主角睡着/醒来 |
| Dissolve 溶解 | ★ | 同一场景氛围转换（白天→傍晚） |
| Movie 全屏视频 | ★★ | 游戏 OP、ED、章节过场动画、重要 CG 演出 |
| 3D Stage 相机推拉 | ★★★ | 电影感开场：从天空俯瞰 → 拉近到校门 → 推到主角背影 |
| Ken Burns 缓动（ATL zoom） | ★★ | 风景 CG、回忆旁白画面，避免画面静止死板 |
| ImageDissolve 特殊形状溶解 | ★★ | 章节标题淡入、从主菜单切入游戏的仪式感切换 |

**选择建议**：普通场景转 `dissolve` 就够了；章节开头配 `fade`；重要章节/高潮配 `Movie` 或 `ImageDissolve`。

## 2. 角色登场 / 退场

| 效果名 | 难度 | 适合场景 |
|---|---|---|
| ATL ease 左/右滑入 | ★ | 常规立绘登场（转学生进教室） |
| MoveTransition 自动插值 | ★ | 角色走位切换（从中间移到左边给新角色让位） |
| 弹跳缓动 ease_bounce | ★ | 活泼角色登场（元气少女蹦出来） |
| 淡入 + 缩放 | ★ | 神秘角色从远处走来 |
| PushMove 推入式 | ★ | 强势角色闯入（"砰！"开门） |
| offscreenleft/right 滑出 | ★ | 角色离开、生气摔门而去 |
| hide + fade | ★ | 角色安静消失、退场回家 |

**选择建议**：角色性格决定登场方式。元气角色弹跳，神秘角色淡入缩放，强势角色推入。

## 使用总原则

1. **克制**：每镜头 1-2 个主要效果，不叠加同类。
2. **服务情绪**：先定情绪（悲/喜/紧张），再选效果。
3. **难度控制**：★★★ 效果需要专业美术/shader 资源，先确认用户能提供再规划。
4. **可跳过**：玩家会按 Ctrl 跳过，所有特效必须能在 skip 模式下正确完成或跳过。
